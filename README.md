# Python Security Tools

This repository contains beginner-level cybersecurity scripts I built while learning penetration testing and network reconnaissance.

## Tools Included

1. Network Device Scanner
Discovers active devices on a local network using ARP requests and retrieves IP and MAC addresses.

2. Port Scanner
Scans a target system to identify open ports and exposed services.

3. FTP Scanner
Checks whether FTP services are available on a target host.

4. IP Geolocator
Retrieves location-related information from IP addresses using an external API.

## Purpose

These tools were built as part of my practical learning journey in ethical hacking and penetration testing using Python.
